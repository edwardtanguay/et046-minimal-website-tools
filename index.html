This site was created with the following simple tools that can help you quickly make decent-looking HTML/CSS sites with minimal custom JavaScript and minimal custom CSS.

The following information, for example, is displayed with Pico.css accordians.

Alpine.js
  • lightweight, JavaScript framework
  • smallest learning curve of any framework
  • it's like Tailwind: all your code lives within valid HTML tags
  • https://alpinejs.dev
Pico.css
  • a clean and lightweight design system
  • less than 10 classes
  • adaptive spacings and typography on all devices
  • light and dark mode
  • https://picocss.com
animate.css
  • a collection of ready-to-use, cross-browser animations that can give your website a subtle animated touch
  • https://animate.style
Emojis
  • search machine for 100s of emojis
  • copy/paste into site without any CDN link
  • https://unicode.party
